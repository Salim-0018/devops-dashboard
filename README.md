# DevOps Dashboard

A colorful DevOps Dashboard built with Node.js and Docker.

## Features

* Dashboard UI
* Dockerized Application
* Node.js Web Server
* Jenkins CI/CD Ready
* GitHub Integration
* Responsive Design

## Tech Stack

* Node.js
* HTML
* CSS
* Docker
* Git
* Jenkins

## Project Structure

```text
.
├── app.js
├── index.html
├── Dockerfile
├── package.json
└── README.md
```

## Run Locally

Install dependencies:

```bash
npm install
```

Start application:

```bash
node app.js
```

Open browser:

```text
http://localhost:3000
```

## Docker

Build image:

```bash
docker build -t devops-dashboard .
```

Run container:

```bash
docker run -d -p 3000:3000 devops-dashboard
```

## CI/CD

This project can be integrated with Jenkins Pipeline for automated build and deployment.
